    Understanding DNA Pattern Search

    KMP Algorithm: The Knuth-Morris-Pratt algorithm efficiently searches for patterns in DNA sequences by avoiding unnecessary character comparisons. It uses a prefix function to skip ahead when mismatches occur.

    Color Coding: Each DNA base is color-coded: A (Adenine), T (Thymine), G (Guanine), C (Cytosine). Matched patterns are highlighted in yellow.

    Tip: Enter a pattern in the search box above to find all occurrences in the sequence. Click on match positions in the sidebar to navigate to them.
